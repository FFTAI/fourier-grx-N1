# 固定位置

## 任务信息

任务值 (TID): **44**

任务描述：

- 机器人保持在当前位置。
- 可先将机器人拖动到目标位置，然后执行此任务，使机器人保持在该位置。
- 需要配合 [执行器使能](/fourier-grx-N1/docs/tasks/servo_on) 任务使用，确保电机处于使能状态。
